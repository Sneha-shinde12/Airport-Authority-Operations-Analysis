# Airport Authority Operations Analysis

## Project Overview

This project analyzes airport operations data to understand flight
movements, operational performance, and trends. The goal is to explore
patterns in airport activities and present insights using data analysis
and visualization tools.

## Objectives

-   Analyze airport operational data
-   Identify patterns in flight operations
-   Visualize operational insights using dashboards

## Tools & Technologies

-   SQL
-   Microsoft Excel
-   Power BI

## Dataset

The dataset includes airport operations information such as flight
activities and operational metrics used to perform analysis and build
the dashboard.
